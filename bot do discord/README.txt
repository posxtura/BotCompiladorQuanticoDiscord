Bot feito pra me familiarizar com python e pra aprender a criar bots no discord com multiplas funções.

Usa as APIs do Hikari e do OpenAI para o funcionamento. Na data de lançamento (no dia que escrevo isso que seria 11/05/2025), pretendo hospeda-lo através do Railway. Não possuo condições para bancar um armazenamento de dados nuvem decente :P

Até então ele só possui duas funções muito básicas, uma delas é a menção (@CompiladorQuantico {Mensagem}) acoplada com o ChatGPT, fazendo com que ele responda como se estivesse participando de uma conversa.
A outra é uma rolagem de dados d100, d20, d10, d8, d6, d4 que funciona com o comando !roll dX (X sendo o número de lados do dado desejado).

Pretendo adicionar mais feats.
